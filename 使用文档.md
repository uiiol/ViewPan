# 外呼数据可视化平台 使用手册

---

## 一、登录

打开浏览器，访问以下地址之一：

| 场景 | 访问地址 |
|------|----------|
| 本机访问 | `http://localhost:80` |
| 局域网访问 | `http://<部署电脑的IP>:80` |
| 固定域名访问 | `http://viewpan.local:80` |

输入管理员分配的用户名和密码，点击登录。

---

## 二、页面结构概览

登录后，系统顶部为导航栏，下方分三个标签页：**数据大盘**、**供应商分析**、**单一客户分析**。

---

## 三、数据大盘（首页）

进入系统后默认显示此页面，用于快速掌握全年整体情况。

**顶部左侧**有年份选择器，可切换查看不同年份（目前支持 2025、2026 年）。

页面包含以下七个模块：

### 3.1 目标完成进度
- **环形进度图**：直观展示当前已完成全年目标的百分比
- **指标说明**：
  - **目标值** = 去年全年通话分钟数 × 1.2（增长 20%）
  - **当前完成** = 今年累计通话分钟数（统计至有数据的最新月份）
  - **完成率** = 当前完成 / 目标值
  - **去年同期** = 去年同期的累计通话分钟数
  - **同比进度** = 今年当前完成 / 去年去年同期（高于 100% 为正增长，低于 100% 为负增长）
- **颜色含义**：绿色 = 领先去年节奏，橙色 = 略落后，红色 = 落后较多

### 3.2 全年达标预测
- 根据当前月均外呼量，预测三种情景下的全年总量：
  - **保守情景**：剩余月份按八折计算
  - **中性情景**：按当前月均不变推算
  - **乐观情景**：当前月均 × 1.1
- 绿色进度条 = 预测能达到目标，红色 = 预计不达标

### 3.3 月度趋势
折线图展示每月实际外呼量：

- **蓝色实线**：今年各月实际通话分钟数
- **灰色虚线**：去年同月通话分钟数（用于同期对比）
- **红色虚线**：全年目标月均线（目标值 / 12）

### 3.4 Top 10 客户
通话分钟数排名前 10 的客户，按柱状图展示。

### 3.5 客户结构
饼图展示三类客户的占比：

- **新客户**：今年有外呼、去年全年无外呼的客户
- **老客户**：今年和去年同期均有外呼的客户
- **流失客户**：去年有外呼、但今年截至目前暂无外呼的客户

### 3.6 渠道分布
各渠道（供应商）的通话分钟数占比，按饼图展示。

### 3.7 客户健康度（散点图）
每个点代表一个客户：

- **X 轴**：去年同月通话分钟数
- **Y 轴**：今年同月通话分钟数
- **绿色点**：今年多于去年（增长）
- **红色点**：今年少于去年（萎缩）
- **蓝色点**：新客户（去年无外呼记录）
- **对角虚线**：y = x，在线以上 = 增长，线以下 = 萎缩

## 四、供应商分析

切换到「供应商分析」标签页，按渠道维度查看数据。

### 4.1 筛选条件
- **年份**：必选
- **季度/月份**：可选，限定查看特定时间段

### 4.2 渠道总览
显示所有渠道的汇总指标，包括：总外呼量、接通量、接通率、外呼分钟数、意向率等。

### 4.3 渠道排行榜
- 支持切换排序指标：**通话分钟数 / 外呼量 / 接通量 / 接通率 / 意向率**
- 指标下方显示中位数和平均值作为参考
- **颜色含义**：
  - 绿色：中位数以上（表现较好）
  - 红色：低于平均数且处于 Q1 水平（需关注）
  - 蓝色：正常水平

### 4.4 月度趋势图
勾选要显示的指标（可多选）：

| 指标 | 说明 |
|------|------|
| 外呼量 | 当月总外呼次数 |
| 通话分钟数 | 当月累计通话时长（分钟） |
| 接通率 | 接通量 / 外呼量 |
| 意向率 | 意向客户数 / 接通量 |

### 4.5 渠道明细表
所有渠道的详细数据，支持翻页浏览。

## 五、单一客户分析

切换到「单一客户分析」标签页，查看单一客户的详细数据。

### 5.1 选择客户
从顶部客户下拉列表中选择目标客户，支持搜索。

### 5.2 客户总览
选中客户的汇总指标（外呼量、接通量、接通率、分钟数、意向率等）。

### 5.3 客户排行榜
所有客户的排名列表，支持切换排序指标（通话分钟数 / 外呼量 / 接通量）。

点击排行榜中的客户行，下方趋势图和分析模块会自动切换到该客户。

### 5.4 月度趋势图
展示该客户各月的趋势，指标含义同渠道分析页面，支持多指标叠加显示。

### 5.5 AI 分析
输入针对该客户的分析问题，AI 将基于数据给出解读。提问越具体，分析结果越有价值。

**示例问题**：
- "这个客户3月的接通率为什么下降了？"
- "对比一下这家客户近3个月的趋势"

## 六、账号管理（仅管理员）

管理员可在顶部导航栏右侧看到「账号管理」入口。

可执行以下操作：

- 查看所有用户列表
- 新增用户（设置用户名、初始密码）
- 修改用户角色（普通用户 / 管理员）
- 删除用户

---

## 七、常见问题

**Q：页面打不开或数据加载空白？**
A：请确认部署电脑的 Docker 服务正常运行。可尝试重启 Docker Desktop。

**Q：排行榜颜色代表什么？**
A：绿色 = 表现优秀（Q3/Q4水平），红色 = 表现较弱（Q1水平），蓝色 = 正常水平。

**Q：想要导出数据怎么办？**
A：请联系管理员，从后台数据库导出。

**Q：忘记密码怎么办？**
A：联系管理员，管理员可在账号管理中重置密码。

## 八、管理员操作指引

### 8.1 新增数据
将新的 Excel 数据文件放入部署电脑的 `原数据` 文件夹，文件名格式无特殊要求。放入后通知管理员执行导入脚本，数据将自动入库。

### 8.2 新增用户
以管理员账号登录系统，点击顶部「账号管理」，点击「新增用户」，填写用户名和初始密码即可。

### 8.3 重置用户密码
在账号管理页面，找到对应用户，点击删除后重新新建即可。
